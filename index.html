Jeu • Godot • Web

Concept

Meteor Escape est un mini‑jeu où le joueur doit esquiver des météorites dans un décor simple et lisible. L’objectif : travailler le rythme, les feedbacks visuels et la gestion des collisions.

Cette version web est un prototype de niveau, jouable directement dans le navigateur ou dans une fenêtre dédiée.

  • Godot
  • Export Web
  • Prototype

Astuce : pour un meilleur confort, joue en plein écran sur ordinateur. Sur mobile, les performances peuvent varier.

Qui je suis et ce que je cherche à construire.
